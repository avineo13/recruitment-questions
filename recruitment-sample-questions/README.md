# Recruitment Sample Questions

This repository contains sample questions used in the recruitment process at Kuberox Technologies.

## Purpose

This repository serves as a collection of sample questions that candidates may encounter during technical interviews and assessments.

## Structure

Questions are organized by topic for easy navigation. The repository contains the following categories:

- **Linux Shell** - Questions related to shell scripting, command-line operations, and Linux utilities
- **Regular Expressions** - Pattern matching and regex-related questions
- **Mathematics** - Mathematical problems and concepts
- **Algorithm** - Questions covering algorithms, algorithm design, and problem-solving
- **Data Structure** - Questions covering data structures and their implementation

Each folder contains sample questions specific to that topic area.

## Contributing

Please follow the repository guidelines when adding new questions or updating existing ones.
